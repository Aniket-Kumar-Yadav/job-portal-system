# 🚀 Job Portal System

A full-stack MERN job portal web application that connects employers and job seekers with role-based authentication, job posting, and application tracking features.
---

## 📌 Features

- 👨‍💼 Employer Registration (Company Email Only)
- 👨‍🎓 Job Seeker Registration
- 🔐 Secure Authentication (JWT + bcrypt)
- 📄 Job Posting System
- 📥 Job Application System
- 📊 Dashboard for Users

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

---

## 🔐 Authentication

- JWT (JSON Web Token)
- Password hashing using bcrypt

---

## 🔄 API Methods

- GET → Fetch data
- POST → Create data
- PUT/PATCH → Update data
- DELETE → Remove data

---

## 👥 User Roles

### Employer
- Post jobs
- Manage applications

### Job Seeker
- Apply for jobs
- Track application status

---
